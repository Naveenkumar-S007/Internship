# Reports Documentation

## Overview

Reports help users analyze and monitor internship activities.

---

# 1. Intern Summary Report

## Purpose

Displays all interns along with their mentor and current status.

## Columns

| Column Name |
| ----------- |
| Intern Name |
| College     |
| Mentor      |
| Status      |

## Sample Output

| Intern Name  | College     | Mentor | Status |
| ------------ | ----------- | ------ | ------ |
| Naveen Kumar | ABC College | John   | Active |

---

# 2. Task Report

## Purpose

Displays tasks assigned to interns.

## Columns

| Column Name |
| ----------- |
| Task Title  |
| Intern      |
| Due Date    |
| Status      |

## Sample Output

| Task Title | Intern       | Status    |
| ---------- | ------------ | --------- |
| Login Page | Naveen Kumar | Completed |

* Generate internship insights
